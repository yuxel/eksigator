Eksigator :: Ekşisözlük Başlık Takip Aparatı
Ekşigator, Ekşisözlük'te takip ettiğiniz başlıklara yeni eklenen entry'leri belli aralıklarla size bildiren bir uygulamadır. Detaylı bilgiyi http://eksigator.com adresinden bulabilirsiniz.
Takip ettiğiniz güncellenen başlıkları Facebook üzerinden de bu uygulama ile takip edebilirsiniz.
Nasıl Kullanabilirim?
  1. Eksigator.com adresindeki Kayıt sayfasından e-posta adresiniz ve belirlediğiniz bir parola ile kaydolun.
  2. E-posta adresinize gelen etkinleştirme bağlantısına tıklayarak üyeliğinizi etkinleştirin.
  3. Etkinleştirme sayfasında sizin için üretilen API anahtarını bir yere not edin. Bu API anahtarı ve e-posta adresiniz, tüm Eksigator uygulamaları için gerekli olan bilgilerdir.
  4. Daha sonra aşağıdaki alana e-posta adresinizi ve API anahtarınızı girin.
  5. Facebook tarafından bilgilendirilme aralığınızı belirleyin.
