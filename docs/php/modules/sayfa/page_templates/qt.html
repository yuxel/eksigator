Qt ile yazılmış C++ istemcisi
Hakkında
Qt kütüphaneleri kullanılarak C++ dili ile Osman Yüksel tarafından geliştirilmiştir.

Ekran görüntülerine ise buradan bakabilirsiniz.
İndirme
http://github.com/yuxel/eksigator-qt/tarball/master adresindeki dosyayı bir dizine açın ve "eksigator" isimli dosyayı çalıştırın.

Not: Şu anda sadece, Linux altında 32bit işlemciler için derlenmiş çalıştırılabili dosya bulunmakta.
Qt derleyicileri olan herhangi bir sisteme sahipseniz (Windows, Mac, Symbian(!)) dizin içine girip "qmake && make" komutlarını vererek dosyayı derleyebilirsiniz.
Yapılandırma
Ekrandaki alana e-posta adresinizi ve API anahtarınızı girip "Kontrol et" düğmesine basarak yeni başlıkları kontrol edebilirsiniz.
Güncelleme
Güncellemeler için aşağıdaki haberler bölümüne göz attıktan sonra, yenilikler için kodu tekrar derlemeniz gerekecektir.
