Google Chrome Ek$igator Eklentisi
Hakkında
Oğuz Kartal tarafından geliştirilmiştir.
İndirme

https://chrome.google.com/extensions/detail/omcnhchbbkgjfolhfejahmfpoahgmpfo?hl=en adresinden yüklenebilir.

Yapılandırma
Eklentiyi kurduktan sonra, sağ üstteki Eksigator logosuna sağ tıklayarak "Seçenekler" menüsünden, Eksigator kullanıcı adınızı ve API anahtareınızı girebilirsiniz
Güncelleme
Güncelleme işlemleri tarayıcınız tarafından otomatik olarak yapılmaktadır.
Kaynak kodu
Kaynak kodları https://github.com/0ffffffffh/EksigatorChromeExtension adresinden edinilebilir.
Haberler
11 Nisan 2011 - 0.9 beta sürümü duyuruldu.
