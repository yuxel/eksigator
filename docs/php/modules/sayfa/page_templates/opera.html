Opera Ek$igator Eklentisi
Hakkında
UserJS kullanılarak Osman Yüksel tarafından geliştirilmiştir.
İndirme
Eklentiyi kuracağınız bir dizin oluşturun. Örn: C:\ekgisagor
Daha sonra http://github.com/yuxel/eksigator/raw/master/api/js/userjs/opera.js dosyasına sağ tıklayarak "Hedefi Farklı Kaydet" diyerek, dosyayı az önce açtığınız dizinin içine kaydedin.
Yapılandırma
Az önce açtığınız dizine gidip, inen dosyayı bir metin editorü ile açın.
Dosya içindeki aşağıdaki satırlardaki demo olan yerleri sırasıyla e-posta adresiniz ve API anahtarınızla değiştirip kaydedin.
var userName = "demo";
var userApiKey = "demo";

Daha sonra Ekşisözlük'ü ziyaret edin. Sayfaya sağ tıklayın ve sırasıyla Site tercihlerini düzenle > Komutlandırma sekmesine tıklayın. Burada bulunan Kullanıcı JavaScript dosyaları dizini yolunu az önce dosyayı kaydettiğiniz dizin olarak değiştirin.
Bundan sonra Ekşisözlük'ü ziyaretlerinizde sağ tarafta Ek$igator'ı görebileceksiniz.
Güncelleme
Güncelleme işlemleri büyük oranda otomatik olarak yapılmaktadır. Çok düşük durumlarda dosyayı tekrar indirip, tekrar yapılandırmanız gerekebilir. Bu tip durumlar için aşağıdaki haberler bölümüne göz atın.
