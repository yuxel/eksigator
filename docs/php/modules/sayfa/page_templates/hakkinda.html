Hakkında
Ek$igator,
- SourPHP kullanılarak geliştirilmiş
- Ekşisözlük'te takip ettiğiniz başlıklardaki değişiklikleri periyodik (1 saat) olarak kontrol ederek size bildiren
- Ekşisözlük yazarı olmayan kullanıcılar tarafından da kullanılabilen bir uygulamadır
İsim Nereden Geliyor?
Ek$igator'ı, bir aggregator olarak düşünerek, bu ismi koyduk. Logo da bir alligator
Nasıl Kullanırım?
Ekşigator'ı kullanabilmek için bir API anahtarına sahip olmalısınız. API anahtarınız, kayıt olduktan sonra size gönderilen bir koddur. Bu anahtarı kullanarak, tarayıcınız için indireceğiniz eklentiyi yapılandırabilirsiniz.
Yazılım Geliştiricler
Ek$igator'ın kaynak kodları açıktır.
Ayrıca, Ek$igator altyapısını kullanarak kendi uygulamanızı geliştirmek isteyen yazılım geliştiriciler için bir JSONP API sunar.
