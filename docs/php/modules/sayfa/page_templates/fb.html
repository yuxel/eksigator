Facebook Uygulaması
Hakkında
Facebook API kullanılarak PHP dili ile Osman Yüksel tarafından geliştirilmiştir.

Ekran görüntülerine ise buradan bakabilirsiniz.
Kurulum
http://apps.facebook.com/eksigator adresini ziyaret edip, Facebook uygulamasını yükleyebilirsiniz.

Uygulamayı kaldırmak için ise http://www.facebook.com/apps/application.php?id=275206958731 sayfasındaki "Uygulamayı Kaldır" bağlantısına tıklayabilirsiniz.
Uygulamayı yüklediğinizde e-posta adresinizi ve API anahtarınızı girererk uygulamayı kurabilirsiniz.
Daha sonra Ayarlar sayfasından, Facebook tarafından bilgilendime aralığınızı seçebilirsiniz.
Belirlediğiniz bu aralıklarla, eğer takip ettiğiniz başlıklara yeni bir entry girilmişse, Facebook tarafından "Bildirim" alacaksınız.
