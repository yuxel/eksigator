Firefox Ek$igator Eklentisi
Hakkında
Murat Çorlu tarafından geliştirilmiştir.
İndirme

https://addons.mozilla.org/en-US/firefox/addon/58195 adresinden "Let me install this experimental add-on." kutusunu işaretledikten sonra "Download now" ile eklentiyi kurabilirsiniz.


Geçici olarak http://api.eksigator.com/js/eksigator-0.1e.xpi adresindeki eklentiyi kullanabilirsiniz.
Yapılandırma
Eklentiyi kurduktan sonra Ekşisözlük'ü ilk ziyaretinizde, size kullanıcı adınız(E-posta adresiniz) ve API anahtarınız sorulacak.
Bunları girdikten sonra Ekşisözlük'ü ilk ziyaretinizde sağ tarafta Ek$igator'ı görebilirsiniz. Daha sonra kullanıcı adınızı veya e-posta adresinizi, "Araçlar -> Eklentiler -> eksigator -> Özellikler" yolu ile değiştirebilirsiniz.
Güncelleme
Güncelleme işlemleri tarayıcınız tarafından otomatik olarak yapılmaktadır.
